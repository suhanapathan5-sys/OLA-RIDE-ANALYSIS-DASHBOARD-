# POWER BI
                                          OLA RIDE ANALYSIS DASHBOARD 
                                          
The OLA Ride Analytics Dashboard is an interactive Power BI project developed to analyze ride booking data and generate meaningful business insights. This dashboard focuses on tracking key performance indicators such as total bookings, successful rides, and cancellations. It also provides a detailed understanding of customer and driver behavior, revenue patterns, and ride trends over time. By using filters, slicers, and visually intuitive charts, the dashboard enables users to explore data dynamically and make informed, data-driven decisions. The project demonstrates strong skills in data cleaning, data modeling, DAX calculations, and dashboard design

**Tools & Skills**

Data Cleaning & Transformation, data modeling, data relationship, 
calculation, columns  & measures, report design, drill down analysis, Slicers & filters, data integration (Excel, SQL, e t c), performance Optimization, data storytelling, publishing  and sharing  reports (POWER BI Service) , raw-level Security, Dashboard automation

**Objective**

Overall, this project highlights my ability to transform raw data into meaningful insights using Power BI. It demonstrates my understanding of data analysis, visualization techniques, and business problem-solving. By building an interactive and user-friendly dashboard, I was able to present complex data in a clear and actionable format, helping stakeholders identify trends, improve decision-making, and enhance overall operational efficiency.
